    Taking user input

    Phew! We have sure come a long way. Let's see how we can take user input in Java

    For this, you need to add an extra import statement 

                
                import java.util.Scanner;

    This Scanner class allows us to accept an input from the standard input(eg: Keyboard).

    In order to take input, you need to create an object of the Scanner class, and then use it to accept an input. Let's observe this from a code example.

                
                import java.io.*;
                import java.util.Scanner;

                public class MyClass
                {
                    public static void main(String args[])
                    {
                        int age;
                        Scanner s=new Scanner(System.in);
                        age=s.nextInt();//Takes user input
                        System.out.println(age);//Returns the input taken.
                    }
                }

    As you saw from the code above, we created an object of the scanner class, and assigned it to the variable

    There are various methods that allow us to take input(These depend on the data type that we are trying to take as input, so select it carefully. I'll list out the common ones below).

    • nextInt(): Takes an integer input from a new line
    • nextFloat(): Takes a floating point input from a new line
    • nextLong(): Takes a long int input from a new line
    • nextDouble(): Takes a double type input from a new line
    • nextLine(): Takes a string/character input from a new line
    What are functions?

    We already had our first encounter with functions, when we learnt about classes and objects. In this section, let's try to understand what a function is all about.

    Simply put, a function is a piece of reusable code, that performs a designated operation for various inputs.

    Example: A function, that takes in two numbers as inputs, and returns the sum of those two numbers.

    Let's now look at how to create functions

                
                int sum(int a,int b)
                {
                    return a+b;
                }

    Okay. So what really happened here?I'll try to dissect it, bit by bit

    So, this little part above, it called function definition(As the name suggests, here, you define what the function does.

    Every function definition contains two parts at the start: The return type(the data type that is being returned by the function, after performing the operations), and the function name.

    Within the curly braces, we then define what the function would do. In our function, we just have a return statement, that sends the value mentioned after the return keyword back to the standard output.

    We also see two numbers being fed as inputs to the function. These are called function arguments.

    Since this function has two arguments, it should be fed two values as inputs, else it'll throw an error.

            {

    Let's loop you in

    Now we come to the last topic of this documentation:Loops

    Loops help us to do a repetitive task. For example, finding the factorial of the first 10 numbers, or summing the first ten natural numbers.

    There are mainly three kinds of basic loops.These include

    • For loop
    • While loop
    • Do-while loop

    Their functionality is more or less similar. The only difference is the way each of them is written. Let's see

    The For loop

    The for loop is the easiest loop to implement. It is controlled by three variables: An initilizer, an update condition, and a change function

    Let's see how a For loop is implemented

            
         //A simple for loop to calculate sum of numbers from 1-10
         int sum=0;
         for(int i=1;i<=10;i++)
         {
             sum=sum+i;
         }
         System.out.println(sum);

    The while loop

    The while loop is a similar entry controlled loop as the for loop. Then only thing that varies is the syntax

    Let's see how a while loop is implemented

     
        
         //A simple program to calculate sum of numbers from 1-10 using a while loop
         int sum=0;
         int i=1//Initializer;
         while(i<=10)//Update condition
         {
             sum=sum+i;
             i++; //Change function
         }
         System.out.println(sum);

    The do-while loop

    The do-while loop is an exit controlled loop. What it means is that it checks the condition on exit(unlike the while or for loop, which tests condition on loop entry). This ensures that the loop executes atleast once, even if the condition is false.

    For the given code, while loop will give value as 10, whereas the for loop and the while loop will give value as 0 

            
            int i=10;
            int sum=0;
            do{
                sum=sum+i;
                i++;
            }while(i < 10);
            System.out.println(sum);

    Let's see how a do-while loop is implemented

     
        
         //A simple do-while loop to calculate sum of numbers from 1-10 
         int sum=0;
         int i=0//Initializer;
         do
         {
             sum=sum+i;
             i++;
         }while(i <= 10);
         System.out.println(sum);
